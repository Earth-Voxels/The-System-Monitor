# About
A system monitor made with Python

# Requirements
-Any Windows NT Operating system

-Any Linux distro

-Python 3.0 +

-psutil

-colorama

# Installing

-install Python 3.0 from python.org

-install psutil and colorama with 

-- "python -m pip install psutil"

-- "python -m pip install colorama"

# Using

Run "system-monitor.py"

and it should work!
